# Practicas-javaScript

Ejercicios personales para dominar JavaScript desde lo básico hasta lo avanzado.

Este repositorio recopila todas las prácticas que realizo diariamente mientras estudio JavaScript. Incluye ejercicios sobre fundamentos del lenguaje, lógica, funciones, callbacks, funciones de orden superior, arrays y más.
Mi objetivo es construir una base sólida que me prepare para temas avanzados como DOM, asincronía y React.

Objetivo del repositorio

Practicar diariamente JavaScript para fortalecer mi lógica, aprender a escribir código limpio y prepararme para trabajar con herramientas modernas del ecosistema web.

Nota

Todos los ejercicios están hechos desde cero para reforzar el aprendizaje real, sin copiar soluciones externas.
